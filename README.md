# Finance-Data-Analysis
Designed an interactive Finance Dashboard in Power BI showcasing Income, Expenses, and Savings trends with KPIs, donut charts, and line visuals. Built DAX measures for monthly growth, savings %, and year-wise insights, enabling clear financial performance tracking.
